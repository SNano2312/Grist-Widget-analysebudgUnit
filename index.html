📈 Courbe d'évolution — Règles de calcul

        

          

            Courbe
            Calcul
          

          
            

              Commandes engagées (cumulées)
              Cumul mensuel des montants dont Texte type val. = "Commandes d'achat" ET Type de montant = "Original". Plafonné au budget à chaque mois.

              Justification : filtrer uniquement les Commandes d'achat Original évite les re-engagements sur Factures et Pièces préenregistrées qui gonflent artificiellement le cumul. Les Réductions et Modifications compensant les Originaux sont exclues.		
            

            

              Budget AE N
              Ligne horizontale fixe = Budget AE N filtré.
            

            

              Prévision
              MIN(dernier cumul + CDa engagées + Reste à réceptionner, Budget AE N). Projetée jusqu'à la date de fin du programme (Table1, Col. C) ou décembre de la dernière année visible si pas de date.
            

            

              Comparaison 2025/2026
              Option "Comparaison 2025/2026" dans le filtre Année. Courbes superposées par mois calendaire (Jan–Déc), une couleur par année. Seules les opérations dont la date comptable appartient à l'année concernée sont incluses.

🏆 Top fournisseurs

        

          Basé sur les Paiements positifs (montants réellement décaissés).

          Source : Texte type val. = "Paiements", montants > 0, colonne Nom 1 = nom du fournisseur.

          Filtrable par Année / Programme / Fonds. Choix du nombre de fournisseurs à afficher (Top 10, 20 ou 50).

          Export CSV disponible via le bouton ⬇️ Export CSV.
        

      


      

        
⏱ Délais de liquidation

        

          Délai = date de la Facture − date de la Commande d'achat d'origine.

          Liaison via le champ Numéro de la pièce précédente de la Facture qui référence le Numéro de pièce de référence de la Commande d'achat.

          Code couleur : Vert ≤30j • Orange 31–90j • Rouge >90j.

          Note : une commande avec plusieurs échéanciers génère plusieurs délais distincts. Export CSV disponible.
        

      


      

        
🔗 Hyperliens N° Commande (Drill-down Histogramme)

        

          Dans le drill-down de l'Histogramme, chaque N° de commande est un lien cliquable qui ouvre directement la commande dans SIFAC+.

          Format de l'URL générée :

          https://inrae.sifacplus.amue.fr/sap/bc/ui2/flp#PurchaseOrder-manage&/C_PurchaseOrderTP(PurchaseOrder='<N°>',DraftUUID=guid'00000000-0000-0000-0000-000000000000',IsActiveEntity=true)
        

      


      

        
🔍 Drill-down Histogramme

        

          Cliquer sur une barre de l'histogramme affiche le détail des lignes de consommation correspondantes (toutes opérations hors Paiements et Comptabilis. coûts secondaires).

          Le tableau est triable par colonne. Les colonnes affichées sont : Année, Date, Type de valeur, N° Commande (lien SIFAC+), Fournisseur, Désignation, Programme, Fonds, Type de montant, Montant.

          Cliquer sur une carte de la Synthèse bascule directement sur l'Histogramme filtré sur ce programme.
