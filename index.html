📂 Sources de données

        

          Budget_2025 / Budget_2026 : données budgétaires par CF/Programme/Fonds.

          Conso_2025 / Conso_2026 : opérations SAP ligne à ligne (type de valeur, montant, fournisseur, date).

          Table Programmes : Col. A = N° programme • Col. B = Acronyme • Col. C = Date de fin.

          Table NACRES 2025 et Table NACRES 2026 : postes de commande SAP par année. Col. A = Code NACRES • Col. E = Type document (OSMA=sur marché) • Col. I = Valeur nette • Col. S = Fonds.
        

      

      

        
🔢 Indicateurs budgétaires

        

          Taux de consommation : Conso AE N ÷ Budget AE N × 100 — Vert ≥90% • Orange ≥50% • Rouge <50%

          Taux d'engagement : MIN((CDa+RAR+OM+NF) ÷ Budget AE N, 100%) — plafonné (budget peut être révisé à la baisse après engagement)

          Reste à engager : MAX(0, Budget AE N − CDa − RAR − OM − NF)

          Prévision finale : MIN(Conso AE + CDa + RAR, Budget AE N)
        

      

      

        
📈 Courbe d'évolution

        

          Commandes engagées : cumul mensuel des Commandes d'achat + Type montant Original, plafonné au budget.

          Prévision : MIN(dernier cumul + CDa + RAR, Budget). Projetée jusqu'à la date de fin (Table Programmes, Col. C) ou décembre.

          Comparaison 2025/2026 : courbes superposées par mois calendaire, filtrées sur l'année de la date comptable.
        

      

      

        
📊 Répartition — Montants nets SAP

        

          Montants nets = somme de tous les types de montant SAP (Original + Réduction + Modification…) par type de valeur.

          💳 Paiements : décaissements effectifs vers les fournisseurs.

          📄 Factures en cours : factures reçues, comptabilisées mais non encore payées.

          🛒 Commandes en cours : solde résiduel des commandes ouvertes non encore livrées ou facturées.

          📝 Pièces préenregistrées : saisies en attente de validation comptable (notes de frais, etc.).

          Total affiché ≈ Conso AE du fichier budget. Écart possible : Prestations internes non incluses dans l'export de détail. Les types "Réservation de fonds" et "Blocage de fonds" sont exclus car ils ne correspondent pas à des dépenses réelles.
        

      

      

        
🧪 Onglet NACRES

        

          Sur marché : type document commence par OSMA

          Hors marché : tous les autres types (OSBC, CA, CAMA...)

          Valeur nette répartie : montant total engagé sur le poste de commande.

          Reste à facturer : part non encore facturée = commande non soldée.

          Commande non soldée : Reste à facturer > 0 — ligne surlignée en orange dans la vue Détail. Filtre "Non soldées" disponible.
